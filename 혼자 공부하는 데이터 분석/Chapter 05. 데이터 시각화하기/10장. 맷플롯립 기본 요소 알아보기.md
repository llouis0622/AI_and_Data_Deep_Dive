# 10장. 맷플롯립 기본 요소 알아보기

# 1. Figure 객체

- 모든 그래프 구성 요소를 담고 있는 최상위 객체
- `figure()` : 그래프 옵션 조절
    - `figsize` : 그래프 크기 튜플로 지정
    - tight 레이아웃 기본 설정 : 그래프 주변에 공백 최소화
    - `bbox_inches` : 타이트 레이아웃 미사용
    - `dpi` : 그래프 확대

# 2. rcParams 객체

- 맷플롯립 그래프의 기본값 관리
- `scatter.marker` : 산점도 마커 모양 바꾸기
- `marker` : 마커 모양 지정

# 3. 여러 개의 서브플롯 출력하기

- 맷플롯립의 Axes 클래스의 객체, 두 개 이상의 축 포함
- `subplots()` : 서브플롯 정의
    - `set_xlabel()`, `set_ylabel()` : 축 지정

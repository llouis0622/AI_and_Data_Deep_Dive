# 02장. 구글 코랩과 주피터 노트북

# 1. 구글 코랩

- 웹 브라우저에서 무료로 파이썬 프로그램을 테스트하고 저장할 수 있는 서비스
- 웹 브라우저에서 텍스트와 프로그램 코드를 자유롭게 작성할 수 있는 온라인 에디터
- 노트북, 코랩 메모장 : 작성한 코랩 파일
- 셀 : 코드 또는 텍스트의 덩어리
- 텍스트 셀 : 코드처럼 실행되는 것이 아님, HTML & 마크다운 혼용 가능
- 코드 셀 : 파이썬 코드를 입력하고 실행할 수 있는 셀

# 2. 노트북

- 구글 클라우드의 가상 서버 사용
- 서버 메모리 12GB, 디스크 공간 100GB
- `.ipynb` 확장자 사용

# 3. 마크다운 서식

- `#` : h1 Tag
- `##` : h2 Tag
- `###` : h3 Tag
- `####` : h4 Tag
- `#####` : h5 Tag
- `** **` : Bold
- `* *`, `_ _` : 기울임
- `~~ ~~` : 취소선
- ` ` : 코드 서체
- `>` : 들여쓰기
- `-` : 목록
- `[]()` : 링크
- `![]()` : 이미지
- `$ $` : LaTeX

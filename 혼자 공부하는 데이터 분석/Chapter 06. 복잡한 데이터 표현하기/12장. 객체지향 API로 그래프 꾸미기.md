# 12장. 객체지향 API로 그래프 꾸미기

# 1. pyplot 방식과 객체지향 API 방식

- pyplot 방식 : matplotlib.pyplot에 있는 함수 사용
- 객체지향 API 방식 : 명시적으로 피겨 객체와 서브플롯 객체를 만들고 이 객체의 메서드 사용

# 2. 그래프에 한글 출력하기
- `font.family` : 기본 폰트 설정
- `rc()` : 한 그룹 내의 여러 설정 동시 지정

# 3. 출판사별 발행 도서 개수 산점도 그리기
- `s` : 마커 크기 지정
- `c` : 산점도 색 지정
- 컬러맵 : 값에 따른 색상 표현
